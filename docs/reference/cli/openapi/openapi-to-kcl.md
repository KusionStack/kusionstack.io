# OpenAPI to KCL

命令

```shell
kclopenapi generate model -f ${your_open_api_spec.yaml} -t ${the_kcl_files_output_dir}
```

示例：

- 输入文件：test_open_api_spec.yaml:

```yaml
definitions:
  v1.TestInt:
    type: object
    properties:
      name:
        type: string
        format: int-or-string
    required:
      - name
    x-kcl-type:
      import:
        package: v1.test_int
        alias: test_int
      type: TestInt
swagger: "2.0"
info:
  title: Kusion
  version: v0.0.2
paths: {}
```

- 命令：

```shell
kclopenapi generate model -f test_open_api_spec.yaml -t ~/
```

- 输出：~/models/v1/test_int.k

```python
"""

    """v1 test int
    """


    name: int | str
```
