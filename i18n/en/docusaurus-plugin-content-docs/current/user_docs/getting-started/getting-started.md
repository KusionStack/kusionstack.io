# Quick Start

Get started quickly with a quick and intuitive experience of [Kusion](/docs/user_docs/intro/kusion-intro) with a quick installation of Kusion, a quick overview of the KCL configuration language, the Kusion model library, and an example of how to quickly deploy [Kubernetes](https://kubernetes.io) with the official [Guestbook](https://kubernetes.io/zh/docs/tutorials/stateless-application/guestbook) via the Kusion tools.
